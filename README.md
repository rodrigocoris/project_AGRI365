# AGRI365 - Plataforma Empresarial para el Sector Agricola

AGRI365 es una plataforma empresarial desarrollada para digitalizar y optimizar procesos internos del sector agricola.
Este repositorio contiene el sitio web de presentacion del proyecto, construido para mostrar su diseno, funcionalidades y evidencia visual de la implementacion.

## Descripcion General

La solucion AGRI365 fue concebida para apoyar la gestion operativa y administrativa de una organizacion agricola, centralizando procesos clave y mejorando la trazabilidad de la informacion.

El sitio web permite:
- Presentar la propuesta funcional de la plataforma.
- Mostrar el enfoque tecnico utilizado en su desarrollo.
- Exponer una galeria visual de pantallas reales del sistema.
- Comunicar identidad de marca y contexto del proyecto.

## Que Muestra La Pagina Web

La pagina fue desarrollada como una vitrina del proyecto y contiene:

- **Hero principal con identidad de AGRI365 y COAGRIN.**
- **Seccion de descripcion funcional y tecnica del sistema.**
- **Tarjetas de caracteristicas tecnicas**, incluyendo:
  - Desarrollo de aplicacion empresarial con Microsoft Power Apps.
  - Diseno de interfaces para compras, ventas, creditos y seguimiento.
  - Automatizacion de procesos con Power Automate.
  - Integracion de datos con SharePoint Lists como backend.
  - Colaboracion con usuarios finales para pruebas y mejora continua.
  - Buenas practicas de usabilidad, productividad y mantenimiento.
- **Galeria de capturas** con 31 pantallas organizadas por modulos.
- **Visualizacion ampliada de imagenes (modal/lightbox)** con navegacion:
  - Click para ampliar.
  - Flechas anterior/siguiente.
  - Navegacion por teclado (izquierda, derecha, ESC).
- **Diseno visual tematico agricola** (paleta verde, iconografia y estilos personalizados).
- **Footer informativo** con referencia a la plataforma y su autoria.

## Como Fue Desarrollada

Este frontend fue construido con un enfoque moderno de componentes reutilizables y experiencia de usuario:

- **Framework:** React
- **Bundler/Tooling:** Vite
- **Lenguajes:** JavaScript (JSX), HTML, CSS
- **Estilos:** CSS personalizado con variables, gradientes, sombras y diseno responsive
- **Recursos visuales:** SVG para iconos y favicon

### Estructura Base

- `src/App.jsx`: Componente principal con contenido, secciones y logica de galeria/modal.
- `src/styles.css`: Estilos globales y tematica visual agricola.
- `public/images/`: Capturas funcionales del sistema AGRI365.

## Objetivo Del Proyecto

AGRI365 busca mejorar la eficiencia operativa de la organizacion mediante la transformacion digital de procesos internos, reduciendo tareas manuales y elevando la calidad de la informacion para la toma de decisiones.
